/assets//assets/... → github.githubassets.com

📖 使用示例

⭐ 下载 Release 文件（直接拼接原地址）

wget https://github.484947.xyz/https://github.com/user/repo/releases/download/v1.0/app.zip

克隆仓库

git clone https://github.484947.xyz/https://github.com/user/repo

下载 Raw 文件

curl https://github.484947.xyz/https://raw.githubusercontent.com/user/repo/main/file.txt

访问 GitHub API

curl https://github.484947.xyz/https://api.github.com/repos/user/repo/releases/latest
💡 Git 全局代理配置（一劳永逸）：
git config --global url."https://github.484947.xyz/https://github.com/".insteadOf "https://github.com/"
配完后所有 git clone https://github.com/... 自动走代理
